## Acesso local

Clone o repositório:

    git clone https://github.com/danpnx/tarefas-ebac-daniel-nunes.git

Acesse a branch referente ao módulo da atividade seguindo o padrão do repositório (*mod + numero-do-modulo*)

Exemplo: 

`git checkout mod05`
